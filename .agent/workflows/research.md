# /research

Workflow für Recherche-Sessions (Tech-Stack, APIs, Design-Patterns).

## Schritte

1. **Fragestellung definieren**
   - Was genau soll evaluiert werden?
   - Welche Kriterien sind wichtig? (Kosten, Performance, DX, DSGVO, etc.)

2. **Recherche durchführen**
   - Context7 als erste Quelle nutzen
   - Web-Recherche für aktuelle Vergleiche
   - Offizielle Docs lesen

3. **Ergebnis dokumentieren**
   Erstelle ein Artefakt unter `/Volumes/Work/AI/__CODING/BENDERGROUP/docs/research/` mit:
   - Fragestellung
   - Evaluierte Optionen (Tabelle)
   - Pro/Contra für jede Option
   - Empfehlung mit Begründung

4. **Masterplan aktualisieren**
   Falls die Recherche eine offene Frage beantwortet → MASTERPLAN.md updaten.

5. **Robert informieren**
   Kompakte Zusammenfassung mit klarer Empfehlung.
